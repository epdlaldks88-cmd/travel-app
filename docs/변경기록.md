## 2026-06-30 - 초기 기획 확정

### 프로젝트

- 공통

### 변경 내용

초기 기획 완료. 두 개의 개인용 React 앱 개발 결정.

### 주요 결정 사항

#### 공통

- **개인용** 사용 목적 확정
- **0원 운영** 원칙
- React + Vite + Tailwind + PWA 스택
- 호스팅: Vercel 또는 Netlify
- AI 페어 프로그래밍(Cursor/Claude Code) 활용

#### 여행 관리 앱

- **모바일 우선** 설계 (폰이 메인 사용 기기)
- 데이터 모델: Trip → Day → Activity 계층 구조
- Activity 타입: 이동/식사/방문지/숙박/기타
- 동기화: **Supabase 자동 동기화**로 결정
- 사진 저장: 원본은 폰 갤러리/Google Photos, 앱엔 썸네일만 (Supabase Storage)
- 지도: Leaflet + OpenStreetMap (Google Maps 아님)
- 요약본 출력: **PNG만** (PDF는 향후 추가 가능)
- 사진 양 기준: 1박2일 약 100장 → 5년치 약 15GB 예상
- AI 커버 이미지: 앱이 프롬프트만 생성, 외부 무료 툴(Bing Image Creator 등) 사용
- AI 커버 이미지를 **Trip 갤러리의 표지로 사용** (시각적 인덱스)
- AI 이미지 스타일 4종: 수채화 / 지브리풍 / 폴라로이드 콜라주 / 미니멀 플랫
- 입력 모드 분리: 여행 중 모드 (간단) / 정리 모드 (상세)
